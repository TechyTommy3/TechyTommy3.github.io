Submit a Spreadsheet!

To send a spreadsheet, send an email to excelnet-directory@googlegroups.com! To protect aginst exposing my actual email, this is a mailing list only I can access, and emails are not replied to.

All spreadsheets are safe to open.

If possible, attach the spreadsheet to the email. If too big to attach, use OneDrive or Google Drive!

Also, please put in the spreadsheet's name and a nickname!

Only xlsx, xlsm, xlsb, xltx, xltm, xls, xlt, xlam, and xla files allowed. Remember that anyone with an internet connection will be able to access the spreadsheet!
